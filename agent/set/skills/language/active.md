# Language: active voice

Prefer active voice over passive in all written content.

Active voice is shorter, clearer, and names the actor.

- "Mount fails" not "mount is failed"
- "Script creates drives" not "drives are created by script"
- "Builder signs packages" not "packages are signed by builder"

Passive is acceptable when the actor is unknown or irrelevant:
"Packages are cached on first build" (who caches is obvious).
