# Language: imperative mood

Use imperative mood in commit messages, changelog entries,
and task descriptions.

- "Add burn pipeline" not "Added burn pipeline"
- "Fix timeout conflict" not "Fixes timeout conflict"
- "Update dictionary" not "Updated dictionary"

Imperative reads as a command: "Apply this commit and it will
add burn pipeline." Past tense reads as a report of what
happened -- commits describe intent, not history.
