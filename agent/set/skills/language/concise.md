# Language: concise

Write tight. Every sentence earns its place or gets cut.

## Rules

- One idea per sentence. Split compound sentences.
- Cut filler words: very, really, basically, actually, simply,
  quite, rather, somewhat, in order to, due to the fact that.
- Prefer short synonyms: use not utilize, show not demonstrate,
  help not facilitate, start not initialize (except code).
- Delete "this means that" bridges -- let the reader connect.
- No throat-clearing introductions ("It is worth noting that").
- Tables and lists over paragraphs when structure fits.
- If removing a sentence changes nothing, remove it.
